# Redemption Tones Website

Official website for **Redemption Tones**, a live sound, touring, and audio services company operating across Europe.

This project is a custom-built static website created to present the company’s services, references, artist/event work, and contact information in a clean, modern, and responsive format.

## Overview

The site is designed to communicate a professional live production brand with a dark visual identity, strong typography, gallery-driven storytelling, and simple navigation.

It includes:

- Home page with branded hero section
- About page
- Services page
- References page
- Image gallery / slider
- Artist and event logo showcase
- Contact modal
- Responsive layout for desktop and mobile
- Search indexing support with sitemap and SEO basics

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

No framework is required. The project is lightweight and easy to deploy as a static site.

## Features

- Responsive navigation with mobile menu
- Custom hero layout and branding
- Gallery slider with arrows, dots, autoplay, and lightbox
- Scroll reveal and staggered animations
- Contact modal UI
- SEO-focused page titles, descriptions, canonical tags, and sitemap support
- Static asset structure for images, icons, fonts, and styles
